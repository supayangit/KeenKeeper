# 👥 KeenKeeper — Keep Your Friendships Alive

## 📌 Project Name
**KeenKeeper**

---

## 🧾 Short Description
KeenKeeper is a responsive web application built with **Next.js and React.js** that helps users track, manage, and maintain meaningful friendships. It allows users to monitor interactions, set relationship goals, and visualize friendship activity through analytics.

---

## 🛠️ Technologies Used

- **Next.js** – App routing, SSR, and page structure
- **React.js** – UI components and state management
- **Tailwind CSS** – Responsive styling and layout design
- **DaisyUI (or component library if used)** – Prebuilt UI components
- **Recharts** – Data visualization (pie charts for interaction analytics)
- **React Icons / Lucide React** – Icons for UI elements
- **React Toastify** – Toast notifications for interactions

---

## ⭐ Key Features

### 1. 👥 Friend Management System
- Displays friends from a structured JSON dataset
- Individual friend detail pages with profile, tags, bio, and status
- Responsive card-based UI for all screen sizes

### 2. ⚡ Interaction Tracking System
- Log interactions like **Call, Text, and Video**
- Automatically updates timeline with date and activity type
- Toast notifications for each interaction action

### 3. 📊 Analytics & Timeline System
- Timeline page showing full interaction history
- Filter system to sort interactions by type (Call / Text / Video)
- Pie chart visualization showing interaction distribution

---

## 📱 Responsive Design
- Fully responsive across **mobile, tablet, and desktop**
