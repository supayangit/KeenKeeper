## 📌 Project Overview

**KeenKeeper** is a modern responsive friendship management web application built with **Next.js and React.js**. It helps users track interactions, manage relationships, and maintain meaningful friendships through structured activity logging and visual analytics.

## 🧾 Description

KeenKeeper enables users to organize and monitor their friendships by tracking interactions such as calls, messages, and video chats. The application provides structured relationship insights, activity timelines, and visual analytics to help users understand their social engagement patterns.

---

## 🛠️ Tech Stack

- **Next.js** – App Router, SSR, and routing architecture  
- **React.js** – Component-based UI development and state management  
- **Tailwind CSS** – Utility-first responsive styling  
- **DaisyUI / UI Library** – Prebuilt component system (if used)  
- **Recharts** – Data visualization and analytics charts  
- **Lucide React / React Icons** – UI icon system  
- **React Toastify** – Notification system for user interactions  

---

## ⭐ Key Features

### 👥 Friend Management System
- Displays friends from a structured dataset
- Individual profile pages with bio, tags, and status
- Clean, responsive card-based layout for all devices

### ⚡ Interaction Tracking System
- Log interactions such as **Call, Text, and Video**
- Automatic timestamped activity tracking
- Real-time toast notifications for user actions

### 📊 Analytics & Activity Timeline
- Complete interaction history timeline
- Filter interactions by type (Call / Text / Video)
- Visual analytics using pie charts for engagement distribution
